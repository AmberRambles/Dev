Semantic HTML

Semantic HTML is a way to add implicit meaning to your code blocks. It allows for greater readability as well as ease of styling

Tag Meaning
header (Note: NOT 'head') Enclosed code will be for the top of the page, AKA the header
main Your page's main content will be enclosed by this tag
article Enclose a single article (full object coding) in these tags (one for each article on your page). Consider using article instead of div, when appropriate.
figure Used with figcaption, this tag is used to encapsulate images and videos
figcaption Used with figure to keep relevent information or credit with the item in the figure (this tag should be a child of figure)
nav Often contains a ul with navigation links
footer Similar to header, this tag encloses the code for the bottom of the document, AKA the footer
output Used to provide the results of a calculation or operation
aside Used for content which may not be related to the page's main content. Ex. quotations, advertisements

Example:

<html> <head>(metadata)</head> <body> <header>(top of document)</header> <main> <h1></h1> <article></article> <article></article> <article></article> </main> <footer> <h3></h3> </footer> </body> </html>
