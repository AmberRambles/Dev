Tables in HTML

There are several requirements for a "successful" table in HTML.

  1. Must be enclosed in <table> tags.
  2. Tables have rows. So we tag our rows <tr>.
  3. Rows contain cells. Cells contain data. We will shorten to 'table data' or <td>.
  4. If we want descriptive headings, they (the whole 'tr') go in <thead>.
  5. The headings themselves go in 'th' instead of 'td'

Those are the essentials anyway. Collumns are added automatically by adding data to rows. If you want a cell to span more than one collumn, you must include colspan as a property within the 'td' with the # of collumns to span as the value.

Code Example for Table

Note: line-breaks have been removed here for ease of coding.
<table><thead><tr><th>Name</th><th>Email</th></tr></thead><tr><td>Tom Thumb</td><td>Example@notreal.com</td></tr><tr><td>Pallas Athena</td><td>summit@olympus.net</td></tr></table>
